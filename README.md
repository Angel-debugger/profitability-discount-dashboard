

# Profitability & Discount Analysis Dashboard

## Overview

This Power BI dashboard analyzes company profitability, discounts, and regional performance using interactive visualizations.

## Key Metrics

* Total Sales
* Total Profit
* Profit Margin %
* Average Discount %

## Dashboard Insights

* West region generated the highest profit
* Technology category showed strongest profitability
* Sales increased significantly over time
* Discounts impacted overall margin performance

## Tools Used

* Power BI
* Data Modeling
* DAX Measures
* Data Visualization

## Features

* KPI Cards
* Regional Slicer
* Bar Charts
* Trend Analysis
* Interactive Filtering
